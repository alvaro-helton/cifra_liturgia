Este é um site de cifras de músicas católicas litúrgicas para missas em língua portuguesa.
A ideia é ser simples e intuitivo: é apresentada uma proposta de repertório adequada ao dia litúrgico.
Na página inicial, há apenas um calendário no qual se destacam os dias em que há repertório disponível. 
O usuário clica no dia desejado e vai direto para as cifras das músicas. Em cada música é possível mudar o tom com apenas um clique e ouvir um trecho do áudio.
Interface simples e limpa, sem propagandas e pop-ups.
Espero que aproveitem!
